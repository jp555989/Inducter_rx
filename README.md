# Inducter_rx
This project involves designing a receiver for wireless power transfer circuit

In this project I am looking to find a way to supply wirelessly with 5V 5A a load of RGB LEDS. It's essential to make a spinning screen work.

I am using 4 same receiver coils together, it's important to keep in mind that one ground plane connects them all together, but their output voltages can't be linked together, each one output voltage is supposed to supply one PCB with array of RGB LEDS.
